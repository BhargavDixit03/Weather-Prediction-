# Weather-Prediction
🌦️ Weather Prediction & Analysis System
📌 Overview

This project is a Weather Analysis and Prediction System that uses real-time weather data and machine learning models to:

Fetch current weather data using API

Analyze historical weather data

Predict rainfall (classification)

Forecast future weather values (regression)

It combines data science + machine learning + API integration to build a smart weather prediction system.

🚀 Features

🌍 Fetch real-time weather data using OpenWeather API

📊 Data preprocessing and cleaning

🌧️ Rain prediction using Random Forest Classifier

📈 Future weather forecasting using Random Forest Regressor

🔄 Uses historical dataset for training

📉 Model evaluation using Mean Squared Error

Scikit-learn
